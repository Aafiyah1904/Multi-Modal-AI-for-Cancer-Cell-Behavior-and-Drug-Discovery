# Multi-Modal AI for Cancer Cell Behavior and Drug Discovery

##  Overview

This project presents a multi-modal AI framework that integrates histopathology image features with drug response data to analyze cancer cell behavior and predict drug effectiveness.

##  Objective

* Understand tumor morphology using image data
* Model drug response using structured datasets
* Combine both modalities for improved prediction

##  Tech Stack

* Python
* Deep Learning (CNN-based feature extraction)
* Graph-based modeling
* Streamlit (for UI)

##  Project Structure

* `models/` – trained models and architectures
* `pipeline/` – data processing and workflows
* `streamlit_app/` – frontend interface
* `notebooks/` – experimentation and analysis
* `knowledge_graph/` – graph-based representations
* `xai/` – explainability components

## 📊 Dataset

Due to size limitations, datasets are not included in this repository.

You can access them here:

* GDSC Dataset: https://www.cancerrxgene.org/
* BreaKHis Dataset: https://www.kaggle.com/datasets/ambarish/breakhis

##  Installation

```bash
pip install -r requirements.txt
```

##  Run the Application

```bash
streamlit run streamlit_app/app.py
```
